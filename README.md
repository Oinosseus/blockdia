# blockdia
Qt widget for I/O block diagrams

# Build Prerequisites

1. [qt](https://www.qt.io/) >= 5.6
2. [doxygen](http://www.stack.nl/~dimitri/doxygen/)
3. [python](https://www.python.org/) >= 3.5

# Build Workflow

Use the ./make.py command to build the library or documentation.

Examples:  
./make -h  
./make build -h  
./make build --debug  
./make all
